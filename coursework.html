MET CS526 Data Structures and Algorithms

This course covers and relates fundamental components of programs. Students use various data structures to solve computational problems, and implement data structures using a high-level programming language. Algorithms are created, decomposed, and expressed as pseudocode. The running time of various algorithms and their computational complexity are analyzed. My assignments and term project can be accessed on my github here.

Grade: B

MET CS601 Web Application Development

  • HTML
  • XHTML
  • Dynamic HTML (DHTML)
  • Cascading Style Sheets (CSS)
  • Dyamic CSS
  • JavaScript
  • AJAX
  • XML
  • JSON

This course focuses on building core competencies in web design and development. It begins with a complete immersion into HTML essentially XHTML and Dynamic HTML (DHTML). Students are exposed to Cascading Style Sheets (CSS), as well as Dynamic CSS. The fundamentals of JavaScript language including object-oriented JavaScript is covered comprehensively. AJAX with XML and JSON are covered, as they are the primary means to transfer data from client and server. My assignments and term project can be accessed on my github here.

Grade: A

MET CS602 Server-Side Web Development

  • PHP
  • MySQL
  • NodeJS
  • MongoDB

The Server-Side Web Development course concentrates primarily on building web applications using PHP/MySQL and Node.js/MongoDB. The course is divided into various modules covering in depth the following topics: PHP, MySQL, Object oriented PHP, PHP MVC, Secure Web applications, Node.js and MongoDB. Along with the fundamentals underlying these technologies, several applications will be showcased as case studies. Students work with these technologies starting with simple applications and then examining real world complex applications. At the end of this course, students would have mastered the web application development on the server-side. My assignments and term project can be accessed on my github here.

Grade: A-

MET CS701 Rich Internet Applciation Development

The Rich Internet Application (RIA) Development course concentrates primarily on building rich client web applications in the browser for desktop and mobile devices. The course is divided into various modules covering in depth the following technologies: HTML5, AngularJS, and Ionic framework. Along with the fundamentals underlying these technologies, several applications will be showcased as case studies. Students work with these technologies starting with simple applications and then examining real world complex applications. At the end of this course, students would have mastered the latest and widely used RIA methodologies.

Course planned to be taken in the Fall 1 semester

DeSales University

Sieve Lab

Sieve lab is a iPadOS application developed by myself and my Senior Coordinating Seminar class during the Spring '19 Semester. I Served as the team leader for the development team; managing both the front-end and back-end teams. Sieve Lab was designed for a local learning company for the use of their first-grade level students and encourages the students to separate the bands of a basket by specific width to let out smaller sized balls while containing larger balls in the basket when the basket is flipped.

Android Dex

Android Dex is a mobile application developed in Android Studio. This application was created in collaboration with one other partner. The application implements a database search of over 150 entries with 6 fields pertaining to each entry. The goal was to mimic the first-generation pokedex from the cartoon series Pokemon. The application utilizes Java, XML, and SQL.
