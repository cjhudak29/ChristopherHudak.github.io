Welcome to my personal portfolio. I am a software industry professional with a career interest in programming and web development. My passion for programming began at age 16 when I wrote a combination of characters, numbers, and symbols to develop my first static web page that read “Hello, World!” From there, I enrolled in courses that enhanced my programming knowledge and completed my Bachelor’s in Computer Science at DeSales University. Today I work in product support as I pursue my Graduate Certificate in Web Application Development at Boston University.

This website serves as the term project for my MET CS601 Web Application Development course at Boston University. This website will showcase development skills acquired through the accelerated six-week course. While today’s web development trends consist of dynamic content, this website will incorporate course concepts and required technologies to deliver a static website.
